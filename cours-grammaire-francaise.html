01
🎯 Introduction · Gratuit

Comprendre les bases — Les parties du discours

Avant de corriger, il faut comprendre. Ce module pose les fondations : qu'est-ce qu'un nom, un verbe, un adjectif, un adverbe ? Vous apprendrez à identifier chaque mot dans une phrase et à comprendre son rôle grammatical. Une étape indispensable pour tout le reste du cours.

📖 6 leçons ⏱ 45 min 🆓 Accès libre
💡 Ce module est accessible gratuitement. Pas besoin de s'inscrire pour commencer — il suffit de lire et d'apprendre !

Leçons du module

Pourquoi apprendre la grammaire ? Beaucoup de gens pensent que la grammaire est réservée aux professeurs ou aux linguistes. En réalité, maîtriser les règles de base vous donne un avantage décisif : vos e-mails, vos messages professionnels et vos documents inspirent immédiatement confiance et crédibilité.

Dans cette leçon, nous posons la question fondamentale : à quoi sert vraiment la grammaire ? La réponse est simple — la grammaire est le code secret qui permet à deux personnes de se comprendre parfaitement à l'écrit. Une faute de grammaire, c'est comme un bug dans un programme : parfois bénin, parfois catastrophique.

Ce que vous apprendrez dans ce cours : identifier les mots et leur rôle, accorder correctement vos phrases, choisir le bon temps verbal, ponctuer avec précision, et éviter les pièges les plus fréquents. Chaque module s'appuie sur le précédent — progressez à votre rythme.

Le nom est le mot qui désigne une personne, un animal, une chose, une idée ou un sentiment. En français, chaque nom possède un genre (masculin ou féminin) et un nombre (singulier ou pluriel).

Types de noms : Noms communs — désignent une catégorie générale (chien, ville, liberté). Noms propres — désignent un individu particulier, toujours avec une majuscule (Congo, Marie, Paris). Noms abstraits — désignent des concepts (bonheur, courage, justice). Noms concrets — désignent des réalités tangibles (table, eau, soleil).

Fonctions du nom dans la phrase : Le nom peut être sujet (Le chien court), objet direct (Il voit le chien), ou complément du nom (la maison du voisin). Identifier la fonction du nom est la clé pour bien accorder le reste de la phrase.

Exercice : Identifiez les noms dans cette phrase : "La joie des enfants illumine la maison familiale." — Réponse : joie, enfants, maison (noms communs) ; familiaux qualifie maison.

Le verbe est le mot central de la phrase : il exprime une action (courir, manger), un état (être, paraître) ou une transformation (devenir, rester). Sans verbe conjugué, il n'y a pas de phrase complète.

Comment reconnaître le verbe ? Le verbe change de forme selon la personne (je mange / tu manges / il mange), le temps (je mange / j'ai mangé / je mangerai) et le mode (indicatif, subjonctif, conditionnel…). Posez la question "Que fait le sujet ?" pour l'identifier.

Verbes d'état vs verbes d'action : Les verbes d'état (être, sembler, paraître, devenir, rester, demeurer) exigent un attribut du sujet qui s'accorde. Ex : Elle semble fatiguée (fatiguée est attribut, accordé avec elle).

À retenir : Trouver le verbe, c'est trouver le cœur de la phrase. Tout le reste — le sujet, les compléments, les accords — tourne autour de lui.

L'adjectif qualificatif apporte une information sur le nom : il le décrit, le précise ou le qualifie. Il s'accorde toujours en genre et en nombre avec le nom qu'il accompagne.

Règle de base : Si le nom est féminin, on ajoute généralement un -e à l'adjectif masculin (grand → grande, petit → petite). Si le nom est pluriel, on ajoute un -s (grand → grands, grande → grandes).

Adjectifs invariables à connaître : Certains adjectifs ne changent jamais de forme : marron, orange, kaki, chic, sympa. Ex : des chaussures marron (pas marrons).

Position de l'adjectif : La plupart des adjectifs se placent après le nom (une voiture rouge), mais certains se placent avant : beau, joli, grand, petit, jeune, vieux, bon, mauvais, gros. Ex : un beau garçon / une jolie fille.

Les pronoms remplacent un nom pour éviter la répétition. Principaux types : personnels (je, tu, il, nous…), possessifs (le mien, le tien…), démonstratifs (celui-ci, cela…), relatifs (qui, que, dont, où).

Les déterminants introduisent le nom. Ce sont les articles (le, la, les, un, une, des), les adjectifs possessifs (mon, ton, son, notre…), les adjectifs démonstratifs (ce, cette, ces) et les numéraux (deux, trois…). Ils s'accordent avec le nom.

Les conjonctions relient des mots ou des propositions. Coordination : et, ou, mais, donc, or, ni, car. Subordination : que, parce que, bien que, lorsque, si, quand. La conjonction de subordination entraîne souvent le subjonctif — nous y reviendrons au Module 3.

Point d'attention : Ne confondez pas ou (choix : café ou thé) et où (lieu ou temps : la ville où je vis).

Quiz de validation — Module 1

Répondez mentalement à ces 5 questions avant de passer au module suivant :

1. Dans la phrase "Le petit chat dort sur le canapé", identifiez le nom sujet, le verbe et l'adjectif. — Sujet : chat · Verbe : dort · Adjectif : petit

2. Accordez l'adjectif : "des chemises (blanc)" — Réponse : blanches

3. Quel est le genre du mot "autonomie" ? — Féminin

4. Remplacez le nom par un pronom : "Marie aime le chocolat" → Elle aime le chocolat / Marie l'aime

5. Quelle conjonction indique un choix ? — ou (sans accent)

5/5 ? Vous êtes prêt pour le Module 2. Marquez ce module comme terminé !

02
📌 Accord · Fondamental

L'accord sujet-verbe — La règle d'or

La faute la plus fréquente en français est l'oubli ou l'erreur d'accord entre le sujet et son verbe. Ce module décortique toutes les situations possibles : sujets multiples, sujets inversés, sujets collectifs, pronoms relatifs. Après ce module, cette erreur n'existera plus pour vous.

📖 5 leçons ⏱ 50 min 🔒 Formation complète

Leçons du module

La règle fondamentale : Le verbe s'accorde toujours avec son sujet en personne et en nombre. Si le sujet est singulier, le verbe est au singulier. S'il est pluriel, le verbe est au pluriel.

Comment trouver le sujet ? Posez la question "Qui est-ce qui + verbe ?" ou "Qu'est-ce qui + verbe ?". Ex : Les enfants jouent → Qui est-ce qui joue ? → les enfants (pluriel) → verbe au pluriel : jouent.

Piège fréquent — le sujet éloigné : "La liste des invités est longue" — le sujet est liste (singulier), pas invités. Ne vous laissez pas piéger par les mots intercalés entre sujet et verbe.

Règle à retenir : Identifiez d'abord le verbe conjugué, puis cherchez son sujet réel. Ignorez les groupes nominaux intercalés.

Quand deux sujets sont reliés par "et", le verbe se met au pluriel : Pierre et Marie sont arrivés.

Quand deux sujets sont reliés par "ou" ou "ni", l'accord dépend du sens. Si les deux sujets sont possibles simultanément → pluriel. Si seulement l'un peut être sujet → singulier. Ex : Ni l'un ni l'autre ne viendra (un seul viendra) vs Le directeur ou ses adjoints seront présents (les deux possibles).

Sujets juxtaposés avec gradation : "Sa gentillesse, sa patience, son dévouement me touchent" → pluriel car plusieurs sujets distincts.

Exception — sujet repris par un pronom de synthèse : "Ton courage, ta ténacité, tout cela me plaît" → singulier car tout cela résume et reprend.

Le sujet inversé apparaît après le verbe, notamment dans les questions et les incises. Ex : "Viendra-t-il ?" — le sujet est il, le verbe reste au singulier. "Que font les oiseaux ?" — sujet : oiseaux, verbe : font.

Les incises sont des expressions comme "dit-il", "répondit-elle", "s'exclama-t-il". Le verbe s'accorde avec le sujet inversé : "Viens !", cria l'enfant.

Tournures impersonnelles : Avec il impersonnel (il faut, il pleut, il reste, il manque), le verbe est toujours au singulier, même si un groupe nominal pluriel suit : "Il reste trois places" (le sujet réel est grammaticalement singulier avec il).

"Tout" : quand il est sujet collectif désignant un ensemble, le verbe est au singulier : "Tout va bien". Mais quand tous est pronom indéfini pluriel : "Ils sont tous venus".

"Chacun / aucun / personne / nul" → verbe toujours au singulier : "Chacun est responsable", "Aucun n'est venu".

"La plupart / la majorité / une partie" → verbe au pluriel si suivi d'un complément pluriel : "La plupart des gens pensent que…". Mais singulier si le groupe est seul : "La plupart était d'accord" (rare).

"Plus d'un" → singulier : "Plus d'un élève a réussi". "Moins de deux" → pluriel : "Moins de deux heures suffiront".

Dictée d'entraînement — Accord sujet-verbe

Lisez cette phrase et identifiez les erreurs d'accord : "Les résultats de l'examen, que tout les élèves attendaient, a été publiés hier soir. Chacun des candidats ont reçu sa note par e-mail."

Corrigé : "Les résultats de l'examen, que tous les élèves attendaient, ont été publiés hier soir. Chacun des candidats a reçu sa note par e-mail."

Erreurs : 1) a été → ont été (sujet = résultats, pluriel) · 2) tout → tous · 3) ont reçu → a reçu (sujet = chacun, singulier).

03
⏳ Conjugaison · Intermédiaire

Les temps verbaux — Quand utiliser quoi ?

Présent, passé composé, imparfait, futur, conditionnel, subjonctif… La richesse des temps français est aussi une source de confusion. Ce module vous donne les clés pour choisir le bon temps selon le contexte, le registre et le sens que vous souhaitez exprimer.

📖 6 leçons ⏱ 55 min 🔒 Formation complète

Leçons du module

Le présent de l'indicatif exprime une action qui se déroule au moment où on parle, mais aussi des vérités générales, des habitudes et des situations permanentes.

Emplois principaux : 1) Action actuelle : "Je lis ce cours en ce moment." 2) Vérité générale : "La Terre tourne autour du Soleil." 3) Habitude : "Il se lève à 6h tous les jours." 4) Présent de narration (style littéraire) : "César arrive à Rome et proclame sa victoire."

Pièges à éviter : Les verbes du 2e groupe (finir, choisir, réussir) prennent -iss- aux 3 personnes du pluriel : nous finissons, vous finissez, ils finissent. Ne confondez pas avec les verbes en -ir du 3e groupe (partir → il part, sans -iss-).

La distinction fondamentale : L'imparfait décrit une action continue, habituelle ou un état dans le passé. Le passé composé exprime une action ponctuelle, achevée, avec un résultat visible dans le présent.

Imparfait : "Quand j'étais enfant, je jouais dehors tous les soirs." — action habituelle, durée non délimitée.

Passé composé : "Hier, j'ai joué au football pendant une heure." — action ponctuelle, achevée, durée précisée.

Les deux ensemble : L'imparfait plante le décor, le passé composé raconte l'événement : "Il pleuvait (décor) quand elle est arrivée (événement)."

Exercice : "Ce matin, je (dormir) encore quand mon téléphone (sonner)." → dormais / a sonné

Futur simple : exprime une action qui aura lieu dans l'avenir, une promesse ou une prédiction. Formation : infinitif + terminaisons -ai, -as, -a, -ons, -ez, -ont. Ex : je partirai, tu finiras, il fera (irrégulier).

Irréguliers importants : être → je serai · avoir → j'aurai · aller → j'irai · faire → je ferai · venir → je viendrai · pouvoir → je pourrai · vouloir → je voudrai · savoir → je saurai.

Futur antérieur : exprime une action future achevée AVANT une autre action future. Formation : futur de être ou avoir + participe passé. Ex : "Quand tu auras fini ton module, tu passeras au suivant."

Règle clé : Après quand, lorsque, dès que, aussitôt que, si le verbe principal est au futur, le verbe subordonné est au futur antérieur (pas à l'imparfait comme en anglais).

Le conditionnel présent s'emploie principalement pour : 1) exprimer une hypothèse ("Si j'avais le temps, j'apprendrais le piano") ; 2) atténuer une demande par politesse ("Je voudrais un café, s'il vous plaît") ; 3) rapporter des informations incertaines ("Il serait malade selon ses collègues").

Formation : Même radical que le futur + terminaisons de l'imparfait : -ais, -ais, -ait, -ions, -iez, -aient. Ex : je parlerais, tu finirais, il ferait.

Attention à la concordance des temps : Après un verbe principal au passé, la subordonnée avec que se met au conditionnel (et non au futur). Ex : "Il a dit qu'il viendrait" (pas viendra).

Le subjonctif exprime le doute, la volonté, le souhait, la crainte, le jugement ou la nécessité. Il apparaît surtout dans les subordonnées introduites par que.

Expressions qui déclenchent le subjonctif : vouloir que, désirer que, falloir que, souhaiter que, avoir peur que, bien que, quoique, pour que, à moins que, avant que. Ex : "Il faut que tu sois là", "Bien qu'il pleuve, je sortirai".

Formation du subjonctif présent : On prend la 3e personne du pluriel de l'indicatif présent, on retire -ent et on ajoute : -e, -es, -e, -ions, -iez, -ent. Ex : ils parlent → parle → que je parle, que nous parlions.

Irréguliers à connaître : être → que je sois · avoir → que j'aie · aller → que j'aille · faire → que je fasse · pouvoir → que je puisse · savoir → que je sache.

Exercice — Choisissez le bon temps

Conjuguez les verbes entre parenthèses au temps qui convient :

1. "Hier, il (pleuvoir) quand je (sortir) du bureau." → pleuvait / suis sorti(e)

2. "Dès que tu (finir) ce module, tu (passer) au suivant." → auras fini / passeras

3. "Il faut que vous (être) présents à 8h." → soyez

4. "S'il (avoir) plus de temps, il (apprendre) la guitare." → avait / apprendrait

5. "Elle m'a dit qu'elle (venir) demain." → viendrait

Score 5/5 ? Excellent ! Vous maîtrisez les temps verbaux essentiels.

04
🎨 Accord · Intermédiaire

L'accord en genre et en nombre — Adjectifs, participes, noms

Masculin ou féminin ? Singulier ou pluriel ? L'accord des adjectifs qualificatifs et des participes passés est l'une des subtilités les plus redoutées du français écrit. Ce module couvre tous les cas, y compris les participes avec "avoir" et les cas d'invariabilité.

📖 5 leçons ⏱ 50 min 🔒 Formation complète

Leçons du module

Le genre des noms est souvent imprévisible en français — il faut les mémoriser. Cependant, certaines terminaisons donnent des indications fiables.

Terminaisons généralement masculines : -isme, -ment, -age, -eau, -ier, -oir → le tourisme, le mouvement, le garage, le tableau, le métier, le couloir.

Terminaisons généralement féminines : -tion, -sion, -eur (abstrait), -ité, -esse, -ance, -ence → la nation, la décision, la douceur, la liberté, la sagesse, la chance, la patience.

Exceptions courantes : un silence, un musée, un lycée (masculins malgré leur fin en -e) ; une eau, une peau (féminins en -eau). Apprenez ces exceptions par la pratique.

Règle générale du pluriel : On ajoute un -s au singulier. Mais de nombreuses exceptions existent.

Terminaisons spéciales : Noms en -al → pluriel en -aux (cheval → chevaux) sauf : bals, carnavals, festivals, récitals. Noms en -eau → pluriel en -eaux (tableau → tableaux). Noms en -eu → pluriel en -eux (jeu → jeux) sauf pneus, bleus.

Adjectifs au pluriel : Même règle — -al → -aux (national → nationaux) sauf bancals, fatals, natals, navals, tonals. Les adjectifs en -eau font -eaux (beau → beaux).

Noms invariables : Certains noms ont la même forme au singulier et au pluriel : un bras / des bras, une voix / des voix, un corps / des corps (terminés par un s, x ou z au singulier).

Règle : Avec l'auxiliaire être, le participe passé s'accorde en genre et en nombre avec le sujet du verbe.

Exemples : "Elle est partie tôt" (sujet féminin singulier → -e). "Ils sont arrivés" (sujet masculin pluriel → -s). "Elles sont revenues" (sujet féminin pluriel → -es).

Verbes toujours conjugués avec être : Les 14 verbes du "mouvement" : aller, venir, partir, arriver, entrer, sortir, naître, mourir, tomber, rester, monter, descendre, retourner, passer + leurs composés. Plus tous les verbes pronominaux (se lever, se coucher, se souvenir…).

Pronominaux : "Elle s'est levée", "Ils se sont disputés". Le participe s'accorde avec le sujet (qui est aussi le COD dans la plupart des cas).

Règle fondamentale : Avec l'auxiliaire avoir, le participe passé s'accorde avec le complément d'objet direct (COD) — mais seulement si ce COD est placé avant le verbe.

Si le COD est après le verbe → pas d'accord : "J'ai mangé des pommes" → mangé reste invariable.

Si le COD est avant le verbe → accord : "Les pommes que j'ai mangées" → le COD que (= les pommes, féminin pluriel) est avant le verbe → -es. "Je les ai mangées" → le pronom les (COD) est avant → accord.

Cas sans accord : Si le verbe n'a pas de COD, ou si le COD est remplacé par en, pas d'accord : "Des pommes, j'en ai mangé" (en ne détermine pas l'accord).

Astuce : Cherchez ce qui est compté ou désigné précisément avant le verbe. Posez la question : "Qu'est-ce que j'ai [verbe] ?" Si la réponse est avant le verbe, accordez.

Exercice — Accordez correctement le participe passé

1. "Les lettres qu'il a (écrire)." → écrites (COD les lettres avant le verbe)

2. "Elle s'est (blesser) au genou." → blessée (pronominal, accord avec le sujet féminin)

3. "Nous avons (voir) un film formidable." → vu (COD après le verbe, pas d'accord)

4. "Ces photos, je les ai (prendre)." → prises (COD les = photos, féminin pluriel, avant)

5. "Des progrès, nous en avons (faire)." → fait (invariable avec en)

05
📝 Orthographe · Intermédiaire

L'orthographe d'usage et les homophones — Ces mots qui sonnent pareil

a / à, ou / où, son / sont, ce / se, leur / leurs, on / ont… Ces homophones sont à l'origine d'une grande majorité des fautes écrites. Ce module vous donne des techniques mémo mnémotechniques simples pour ne plus jamais les confondre.

📖 5 leçons ⏱ 45 min 🔒 Formation complète

Leçons du module

Un homophone est un mot qui se prononce de la même façon qu'un autre mot mais qui s'écrit différemment et a un sens différent. Ex : mer / mère / maire, tous se prononcent [mɛʁ].

Deux catégories : Les homophones lexicaux (mots du dictionnaire différents : verre / ver / vers / vert) et les homophones grammaticaux (mots de catégories grammaticales différentes : a / à, ou / où, son / sont).

Pourquoi les confondre est grave ? Une confusion d'homophone grammatical change la nature grammaticale du mot et peut rendre la phrase incorrecte ou ambiguë. C'est le type de faute le plus repéré en lecture professionnelle.

Bonne nouvelle : Il existe des astuces simples et efficaces pour distinguer chaque paire — c'est ce que vous verrez dans les leçons suivantes.

a / à — a est le verbe avoir (on peut le remplacer par avait). à est une préposition (on ne peut pas). Ex : "Il a faim" → avait faim ✓. "Il va à Paris" → avait Paris ✗ → préposition.

ou / où — ou signifie "ou bien" (on peut remplacer par ou bien). où est un adverbe de lieu ou un pronom relatif. Ex : "Café ou thé ?" → ou bien ✓. "La ville où je vis" → lieu.

son / sont — son est un adjectif possessif (devant un nom). sont est le verbe être (on peut remplacer par étaient). Ex : "Ils sont là" → étaient là ✓.

ce / se — ce est un déterminant ou pronom démonstratif (ce livre, c'est). se est un pronom réfléchi (il se lave). ces / ses — ces est démonstratif pluriel, ses est possessif pluriel.

leur / leurs — leur (sans s) est pronom COI invariable (je leur parle). leurs (avec s) est adjectif possessif pluriel (leurs amis).

Technique de substitution : Pour tout homophone grammatical, essayez de le remplacer par un équivalent dont vous connaissez la forme. Si ça passe → c'est le bon. Ex : a/à → testez avait. on/ont → testez avaient.

Mémo visuel : Les homophones avec accent (à, où, là, ça, dès, dû) ont généralement une fonction grammaticale adverbiale ou prépositionnelle. Sans accent → souvent verbe ou déterminant.

Astuce pour "leur/leurs" : Remplacez par lui. Si ça fonctionne → leur sans s (pronom). Sinon, c'est l'adjectif possessif → accordez avec ce qui suit. Ex : "Je lui parle" → je leur parle ✓.

Astuce pour "ou/où" : Remplacez par ou bien. Si la phrase garde son sens → ou sans accent. Sinon → où avec accent.

Mots souvent mal orthographiés à mémoriser :

Parmi (sans s) · davantage (en un mot, sans accent) · peut-être (avec trait d'union) · quelque chose (deux mots) · cependant, néanmoins, toutefois (sans trait d'union) · malgré (sans s) · désormais (en un mot) · plutôt (en un mot, avec accent) vs plus tôt (deux mots).

Confusion fréquente : plutôt = préférence ("Je préfère le café plutôt que le thé"). plus tôt = antériorité temporelle ("Il est arrivé plus tôt que prévu").

Règle des adverbes en -ment : Formés sur l'adjectif féminin + -ment. Ex : lent → lente → lentement, doux → douce → doucement. Exceptions : vraiment, absolument, précisément (sur l'adjectif masculin en -i ou -u).

Mini-dictée — Homophones

Complétez avec le bon homophone : a/à · ou/où · son/sont · leur/leurs

"Les élèves (son/sont) __________ dans la salle (ou/où) __________ ils travaillent. Chacun (a/à) __________ rangé (son/sont) __________ cahier. La professeure (leur/leurs) __________ explique (leur/leurs) __________ exercices."

Corrigé : sont · où · a · son · leur · leurs

Vérifications : sont (= étaient ✓) · où (lieu, pas "ou bien") · a (= avait ✓) · son (possessif avant cahier) · leur (= lui explique ✓) · leurs (possessif pluriel avant exercices)

06
✒️ Syntaxe · Avancé

La ponctuation et la syntaxe — Structurer pour convaincre

Une virgule mal placée peut changer le sens d'une phrase entière. Ce module vous enseigne les règles de ponctuation (virgule, point-virgule, deux-points, tiret, guillemets), la construction des propositions subordonnées et l'art de rédiger des phrases complexes mais lisibles.

📖 5 leçons ⏱ 55 min 🔒 Formation complète

Leçons du module

La virgule est le signe de ponctuation le plus utilisé et le plus souvent mal employé. Elle marque une pause courte et sépare des éléments de même nature.

Emplois corrects : 1) Séparer des éléments juxtaposés dans une énumération : "Il achète des pommes, des bananes, des oranges et des poires." (pas de virgule avant le dernier et dans les listes simples). 2) Encadrer une incise ou un complément circonstanciel déplacé : "Hier matin, il a appelé."

INTERDITS absolus : Ne jamais mettre de virgule entre le sujet et son verbe : ❌ "Le chien, aboie fort". Ne jamais mettre de virgule entre le verbe et son COD : ❌ "Il mange, une pomme".

Virgule et conjonctions : On ne met généralement pas de virgule devant et, ou, ni dans une liste simple. Mais on en met une si la proposition est longue et complexe : "Il travaille dur tous les jours, et ses résultats le prouvent."

Le point-virgule ( ; ) sépare deux propositions indépendantes mais liées par le sens. Il est plus fort que la virgule, moins fort que le point. Ex : "Il travaille dur ; ses efforts portent leurs fruits."

Les deux-points ( : ) introduisent une explication, une énumération ou une citation. Ex : "Il a tout expliqué : le pourquoi, le comment et le résultat." · "Il a conclu : 'La grammaire, ça s'apprend.'"

Le tiret cadratín ( — ) s'utilise dans les dialogues pour marquer les prises de parole, et dans les incises pour isoler une information secondaire. Ex : "Il répondit — sans hésiter — que oui." Ne pas confondre avec le trait d'union (-) qui unit des mots (peut-être, c'est-à-dire).

Une proposition subordonnée est une proposition qui dépend d'une autre (la principale) et qui joue un rôle dans la phrase : sujet, COD, complément circonstanciel…

Types principaux : Relative : introduite par qui, que, dont, où → qualifie un nom. "Le livre que tu m'as prêté est passionnant." Complétive : introduite par que → COD du verbe principal. "Je pense que tu as raison." Circonstancielle : exprime le temps, la cause, le but, la concession… "Bien qu'il pleuve, nous sortirons."

Pièges syntaxiques : Ne commencez pas une phrase par "Parce que" seul (c'est une subordonnée sans principale). Évitez l'accumulation de propositions relatives : "le livre que l'auteur que tu connais a écrit..." — préférez reformuler.

Les guillemets français sont les guillemets pointés : « … ». Ils encadrent les citations, les dialogues et les mots employés avec un sens particulier. On met une espace insécable après « et avant ». Ex : « Bonjour », dit-il.

Les parenthèses ( ) encadrent une information secondaire, une précision ou un exemple. Ce qu'elles contiennent peut être supprimé sans changer le sens global. Ex : "Ce phénomène (décrit au chapitre 3) est bien documenté."

Les crochets [ ] s'utilisent dans les citations pour indiquer une modification ou une omission du texte original. Ex : "Il [l'auteur] affirme que..." ou "[…]" pour signaler une coupure.

Note typographique : En français, le point final se place après le guillemet fermant si la phrase principale continue, mais avant si la citation est une phrase complète autonome.

Exercice — Ponctuez et reformulez

Ajoutez la ponctuation manquante et reformulez si nécessaire :

1. "il a réussi son examen il peut être fier de lui" → "Il a réussi son examen ; il peut être fier de lui."

2. "Elle aime plusieurs choses la lecture la musique la cuisine et le sport" → "Elle aime plusieurs choses : la lecture, la musique, la cuisine et le sport."

3. "Le directeur dit la réunion est annulée" → "Le directeur dit : « La réunion est annulée. »"

4. "Bien qu'il soit tard nous devons finir ce travail" → "Bien qu'il soit tard, nous devons finir ce travail."

07
🚩 Pièges · Avancé

Les pièges les plus fréquents — Erreurs que tout le monde fait

Même les bons scripteurs tombent dans certains pièges récurrents. Ce module est une liste commentée des 30 erreurs les plus fréquentes en français écrit : confusion de niveaux de langue, barbarismes, solécismes, pléonasmes et tours mal construits.

📖 4 leçons ⏱ 40 min 🔒 Formation complète
🚩 Ce module est particulièrement utile pour les rédacteurs, journalistes, entrepreneurs et toute personne qui communique par écrit dans un contexte professionnel.

Leçons du module

Un barbarisme est un mot qui n'existe pas en français ou qui est utilisé à tort. Un anglicisme est un emprunt inutile à l'anglais quand un équivalent français existe.

10 à éviter absolument : ❌ "malgré que" + indicatif → ✅ "bien que" + subjonctif ou "malgré" + nom. ❌ "au jour d'aujourd'hui" (pléonasme) → ✅ "aujourd'hui". ❌ "pallier à" → ✅ "pallier" (transitif direct). ❌ "au niveau de" (abus) → ✅ "en ce qui concerne, pour". ❌ "opportunité" au sens de occasion → usage à surveiller. ❌ "compléter un formulaire" (anglicisme) → ✅ "remplir un formulaire". ❌ "finaliser" → ✅ "achever, terminer" selon le contexte. ❌ "avérer vrai" (pléonasme, s'avérer = se révéler) → ✅ "s'avérer". ❌ "suite à" → ✅ "à la suite de, en raison de". ❌ "prérequis" (calque anglais) → ✅ "condition préalable, prérequis" (accepté en contexte pédagogique).

Un pléonasme est une répétition inutile dans laquelle deux mots expriment la même idée. Il alourdит le style et trahit une confusion sur le sens des mots.

Pléonasmes fréquents à éviter : ❌ "monter en haut" (monter = aller en haut) → ✅ "monter". ❌ "descendre en bas" → ✅ "descendre". ❌ "prévoir à l'avance" (prévoir = anticiper) → ✅ "prévoir". ❌ "joindre ensemble" → ✅ "joindre". ❌ "une petite maisonnette" (-ette = petit) → ✅ "une maisonnette". ❌ "comparer entre eux" → ✅ "comparer". ❌ "collaborer ensemble" → ✅ "collaborer".

Cas tolérés en langue courante : "monter en haut" est parfois accepté à l'oral pour l'emphase, mais à éviter absolument à l'écrit professionnel.

Trois niveaux de langue coexistent en français : le niveau familier, le niveau courant et le niveau soutenu. Le contexte détermine lequel utiliser.

Niveau familier → conversations informelles, messages entre amis : "T'as vu ? C'est ouf. On se taille." Évitez en contexte professionnel.

Niveau courant → usage standard, presse, e-mails professionnels : "Avez-vous vu ? C'est impressionnant. Nous partons." C'est le registre de référence.

Niveau soutenu → écrits officiels, littérature, discours : "Avez-vous remarqué ? Cela est remarquable. Nous nous en allons." Passé simple, subjonctif plus présents, vocabulaire riche.

Erreur fréquente : Mélanger les registres dans un même document. Un rapport professionnel ne doit pas contenir d'expressions familières, même entre guillemets.

Exercice — Corrigez ce texte

Texte à corriger : "Suite à notre réunion d'hier matin, je vous envoie en pièce-jointe le rapport que nous avons prévu à l'avance. Malgré que les résultats soit bons, au niveau de la communication, on a des améliorations à apporter. Veuillez compléter le formulaire ci-joint et le retourner en retour."

Corrigé commenté :

✅ "À la suite de notre réunion d'hier" (suite à → à la suite de) · ✅ "en pièce jointe" (sans trait d'union) · ✅ "le rapport que nous avons établi" (prévoir à l'avance = pléonasme) · ✅ "Bien que les résultats soient bons" (malgré que + subjonctif → bien que + subjonctif ; soit → soient) · ✅ "en matière de communication" (au niveau de = abus) · ✅ "remplir le formulaire" (compléter = anglicisme) · ✅ "le retourner" (retourner en retour = pléonasme).

Vous pouvez aussi vérifier vos textes avec Antidote ↗ ou l'outil gratuit BonPatron ↗ pour détecter vos erreurs en temps réel.

🏆 Évaluation finale · Certificat

Exercices pratiques et évaluation finale — Obtenez votre certificat

Le dernier module est entièrement consacré à la mise en pratique. Vous rédigez, corrigez et reformulez des textes complets. À la fin, vous passez l'évaluation finale et, si vous réussissez, vous obtenez votre certificat MPOUCK de maîtrise de la grammaire française.

📖 4 leçons ⏱ 60 min 🎓 Certificat inclus
🎓 Score minimum requis pour l'obtention du certificat : 75 % de bonnes réponses. Vous pouvez repasser l'évaluation autant de fois que nécessaire.

Leçons du module

Récapitulatif des 7 modules :

Module 1 — Identifier noms, verbes, adjectifs, pronoms, déterminants, conjonctions dans toute phrase.

Module 2 — Accorder le verbe avec son sujet réel, même éloigné ou inversé. Gérer les sujets collectifs et multiples.

Module 3 — Choisir le bon temps verbal : imparfait vs passé composé, futur antérieur, conditionnel, subjonctif.

Module 4 — Accorder participes passés avec être (toujours) et avec avoir (seulement si le COD précède).

Module 5 — Distinguer les homophones grammaticaux (a/à, ou/où, son/sont, leur/leurs) par substitution.

Module 6 — Ponctuer correctement (virgule, point-virgule, deux-points) et construire des propositions subordonnées lisibles.

Module 7 — Éviter barbarismes, anglicismes et pléonasmes. Maintenir la cohérence du niveau de langue.

Repassez sur les leçons où vous avez hésité avant de passer à l'évaluation finale.

Dictée de certification — Niveau intermédiaire

Faites écrire ce texte à quelqu'un ou dictez-le vous-même après l'avoir mémorisé :

"Les résultats que l'équipe a obtenus sont remarquables. Bien que les conditions aient été difficiles, chacun s'est montré à la hauteur. La directrice, dont les efforts ont été décisifs, a remercié ses collaborateurs. Désormais, ils peuvent être fiers du chemin parcouru."

Points de vigilance : obtenus (accord COD avant : les résultats) · aient été (subjonctif passé après bien que) · s'est montré (accord avec chacun, singulier) · dont (pronom relatif, pas duquel) · parcouru (invariable : pas de COD avant avoir).

Barème : 1 point par mot correctement orthographié et accordé. Score maximal : 50 points (texte complet).

Exercice de rédaction — Corrigez cet e-mail professionnel

Texte original (avec erreurs) : "Cher Monsieur, suite à votre demande, je vous envoies en pièce-jointe les documents que vous avez demandé. Je reste disponible si vous auriez des questions. Cordialement."

Version corrigée : "Cher Monsieur, à la suite de votre demande, je vous envoie en pièce jointe les documents que vous avez demandés. Je reste disponible si vous avez des questions. Cordialement."

Erreurs corrigées : 1) suite à → à la suite de · 2) envoies → envoie (je, 1re personne : pas de -s aux verbes en -er) · 3) pièce-jointe → pièce jointe (deux mots, sans trait d'union) · 4) demandé → demandés (accord : COD les documents avant l'auxiliaire) · 5) si vous auriez → si vous avez (après si conditionnel, on n'utilise jamais le conditionnel dans la subordonnée).
